# 🦉 Discord Бот

Полнофункциональный Discord бот с модульной архитектурой для администрирования сервера, управления семьями, розыгрышами и мониторинга статистики.

## 📋 Функции

### 🤖 Возможности бота
- **Временные голосовые каналы**: Автоматическое создание войсов с панелью управления (слоты, приватность, переименование, передача владельца)
- **Система заявок в семью**: Создание заявок с интерактивными кнопками и модальными окнами
- **Розыгрыши (Giveaways)**: Система конкурсов с кнопкой участия, требованиями к ролям и автоопределением победителей
- **Логирование**: Отслеживание входа/выхода пользователей с сервера
- **Администрирование**: Команды для модерации и управления сервером
- **Экономика**: Система баланса и уровней с прогрессией XP
- **Профили пользователей**: XP, репутация, статистика сообщений, время в голосе

### 🛡️ Модерация
- Фильтр матерных слов (настраиваемый список для каждого сервера)
- Очистка сообщений (purge)

## 🚀 Установка

### Требования
- Python 3.11+
- MySQL/MariaDB
- Discord.py 2.3+

### 1. Установка зависимостей Ubuntu
```bash
sudo apt update && sudo apt upgrade
sudo apt install apache2 mariadb-server mariadb-client php libapache2-mod-php php-mysql php-redis php-zip phpmyadmin
sudo systemctl restart apache2
```

### 2. Настройка базы данных
```sql
-- Вход в MySQL
sudo mysql

-- Создание пользователя
CREATE USER 'serveradmin'@'%' IDENTIFIED BY 'ваш_пароль';
GRANT ALL PRIVILEGES ON *.* TO 'serveradmin'@'%';
FLUSH PRIVILEGES;

-- Настройка удалённого доступа (опционально)
# Отредактируйте /etc/mysql/mariadb.conf.d/50-server.cnf
# Закомментируйте строку: bind-address = 127.0.0.1
```

### 3. Установка Python-зависимостей
```bash
pip install -r requirements.txt
```

## 📚 Команды бота

### 👤 Профиль и экономика
| Команда | Описание | Права |
|---------|----------|-------|
| `!профиль [@упоминание]` <br> *aliases: `!проф`, `!profile`* | Просмотр профиля пользователя (XP, уровень, баланс, время в голосе, сообщения) | Все |
| `!setbalance <@user> <сумма>` | Установка баланса пользователю | Owner (владелец бота) |

### 🛡️ Модерация
| Команда | Описание | Права |
|---------|----------|-------|
| `!clear <количество>` <br> *aliases: `!purge`, `!очистить`* | Очистка сообщений (до 100) | Manage Messages |
| `!add_banned_word <слово>` | Добавить слово в чёрный список | Administrator |
| `!remove_banned_word <слово>` | Удалить слово из чёрного списка | Administrator |
| `!list_banned_words` | Показать список запрещённых слов | Administrator |
| `!toggle_profanity` | Включить/выключить фильтр мата | Administrator |
| `!profanity_stats` | Статистика срабатываний фильтра | Administrator |

### 🗣️ Каналы приветствий и прощаний (события)
| Команда | Описание | Права |
|---------|----------|-------|
| `!setwelcome <канал>` | Установка канала для приветствий новых участников | Administrator |
| `!setgoodbye <канал>` | Установка канала для прощаний участников | Administrator |

*Примечание: События приветствия и прощания работают автоматически при входе/выходе пользователей*

### 🎤 Временные голосовые каналы
| Команда | Описание | Права |
|---------|----------|-------|
| `!setup_voice <голосовой> <текстовый>` | Настройка системы временных каналов (отслеживаемый канал и канал уведомлений) | Administrator |
| `!voice_stats` | Статистика созданных временных каналов | Все |

*Примечание: Временные каналы создаются автоматически при входе в отслеживаемый канал. Формат: 🦉 | Голосовой ⁰, 🎤 | Голосовой ¹ и т.д.*

### 👨‍👩‍👧‍👦 Система заявок в семью
| Команда | Описание | Права |
|---------|----------|-------|
| `!setup_family <заявки> <рассмотрение> <роль> <аудит>` | Настройка системы заявок (канал подачи, канал рассмотрения, роль для принятых, канал аудита) | Administrator |
| `!family_status` | Статистика системы заявок (принято, отклонено, ожидает) | Administrator |
| `!family_applications [@user]` | Список ожидающих рассмотрения заявок (можно указать пользователя для фильтрации) | Administrator |

*Примечание: Кнопка подачи заявки создаётся автоматически после настройки системы*

### 📨 Дополнительные команды
| Команда | Описание | Права |
|---------|----------|-------|
| `!sendmessage <#канал> <текст>` <br> *aliases: `!sendmsg`, `!msg`* | Отправка сообщения в указанный канал (можно с вложением) | Administrator |

### 🎁 Розыгрыши (Giveaways)
| Команда | Описание | Права |
|---------|----------|-------|
| `!giveaway create` <br> *aliases: `!gw create`* | Создать новый розыгрыш через модальное окно (приз, длительность, кол-во победителей, требования к ролям) | Manage Guild |
| `!giveaway list` <br> *aliases: `!glist`* | Показать список активных розыгрышей на сервере | Все |
| `!giveaway end <ID>` <br> *aliases: `!gend <ID>`* | Завершить розыгрыш вручную и выбрать победителей | Manage Guild / Создатель |
| `!giveaway cancel <ID>` <br> *aliases: `!gcancel <ID>`* | Отменить активный розыгрыш и удалить сообщение | Manage Guild / Создатель |
| `!giveaway reroll <ID>` <br> *aliases: `!greroll <ID>`* | Перевыбрать победителя из участников завершённого розыгрыша | Manage Guild / Создатель |

*Примечание: Розыгрыши поддерживают требования к ролям, подписке и верификации участников. Участники нажимают кнопку "🏆 Участвовать" для вступления.*

## 🗄️ Структура базы данных

### Таблицы, используемые ботом

| Таблица | Описание | Где используется |
|---------|----------|------------------|
| `users` | Пользователи Discord (XP, уровень, баланс, статистика) | `profile.py`, `events.py`, `mysql_connector.py` |
| `server_settings` | Настройки серверов (каналы приветствий/прощаний, каналы для временных войсов) | `admin.py`, `events.py`, `tempvoice.py`, `mysql_connector.py` |
| `voice_channels` | Временные голосовые каналы (ID канала, владелец, название) | `tempvoice.py` |
| `family_applications` | Заявки в семьи (статус, пользователь, дата) | `family_applications.py` |
| `family_settings` | Настройки системы заявок (каналы, роль, аудит) | `family_applications.py` |
| `profanity_words` | Запрещённые слова для фильтрации мата (на сервер) | `profanity_filter.py`, `mysql_connector.py` |
| `server_profanity_settings` | Настройки фильтра мата для серверов | `profanity_filter.py`, `mysql_connector.py` |
| `giveaways` | Розыгрыши (приз, победители, участники, статус, длительность) | `giveaway.py`, `mysql_connector.py` |

*Примечание: Таблицы `families`, `family_members`, `temp_voice_channels`, `servers`, `user_profiles` не используются в текущей версии бота.*
